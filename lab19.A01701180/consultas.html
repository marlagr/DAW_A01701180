PREGUNTAS Y AJAX

Se utilizo ajax en las consultas de esta pagina, para actualizar en tiempo real en registrar evento y para registrar los datos

¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications?
Mejora la experiencia del usuario, lo cual significa que es una aplicación más completa, donde sus FUNCIONALIDADES estan bien comunicadas entre si.

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?
Que la introducción de datos es mas directa, por lo tanto los datos deben de ser validados cuando se hace el registro en la base de datos. Puede ser con la funcion bind_param() o real_escape_string()

¿Qué es JSON?
Es una tecnica usada para obtener un formato desde JavaScript. Se puede usar para validar ydar formato a los datos que entran y salen.
