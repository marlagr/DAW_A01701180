URL ( Uniform Resource Locator) es una extensión identificadora de una dirección en el internet
MIME(Multipurpose Internet Mail Extensions) es el formato estandarizado para escrir las extensiones de un URL[1]
XML(Extensible Markup Language) es un formato estructurado de etiquetas útil para facilitar la lectura de la información mediante algún programa[3]
XHTML Es un leguaje de etiquetas basado en XML
HTML5 Ultima versión de HTML con nuevas etiquetas
JavaScript Lenguaje de programación que se puede utilizar en conjunto con una pagina de HTML
PHP(Hypertext Preprocessor) Lenguaje de desarrollo de paginas web
JSP(JavaServer Pages) Es un lenguaje de programación que utiliza java y va orientado a paginas web
AJAX(Asynchronous JavaScript And XML) Es una técnica que se usa para comunicarse con el servidor sin afectar al web browser
¿Cuál es la diferencia entre Internet y la World Wide Web? El internet hace referencia a las conexiones que hacen posible la comunicación y la World Wide Web es el protocolo hipertextos por el que funciona la internet
¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE? Get: solicitar información o recurso, es el default de la barra de navegación Head: manda información del encabezado (comentario de cómo esta la información) Post: para mandar información al servidor Put: actualizar totalmente información de un registro Patch: actualizar parcialmente un registro Delete: eliminar un registro del servidor
¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué? POST, por que mantiene la información privada y no se envía directo a los servidores como con GET
¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL? El método GET que pide información publica al servidor
Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error? Todo esta bien, lo que se busca recibir
¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué? El recurso que se busca no esta servidor, si esta accediendo a información publica es necesario revisar el error, si no es de dominio publico pues no.
¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué? Si, es un error en el desarrollo de las funciones del sistema
¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados. Significa que para la versión más actualizada ha sido modificado o eliminado el atributo. -->Figure/figcaption en lugar de img -->'i' ahora representa icono en vez de volver la letra italic 'em' --> “bgcolor” en las tablas -->height and width para atributo de las tablas -->“background” en body -->“accept” para los formularios
¿Cuáles son las diferencias principales entre HTML 4 y HTML5? La sintaxis de la estructura con etiquetas más simples, en html5 ahora tenemos la posibilidad de tener páginas mejor ordenadas y con elementos que hagana más facikl su lectura (Section, header, footer, aside, nav, article), en vez de solo los divisores que llegaban a ser confusos.
¿Qué componentes de estructura y estilo tiene una tabla? "table" para encerrar el área de la tabla "thead" para encerrar el área de un encabezado "th" celdas pertenecientes al thead "tr" table row "td" table cell "tfoot" para encerrar el área del pie de la tabla "caption" titulo de la tabla "colgroup" una agrupación de columnas para darles otro formato "col" especifica las propiedades de un colgroup
¿Qué tanto soporte HTML5 tiene SAFARI?  En los formularios o distintos elementos de formularios, como pop-ups o el calendario. Ciertos audios, objetos interactivos, html imports, custom elements.
